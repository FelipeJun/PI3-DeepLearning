Traffic Signs Dataset

O que é Deep learning?

O deep learning tem como base as redes neurais, que são tecnologias capazes de imitar o comportamento do cérebro humano, gerando ações dinâmicas com a captação de diferentes dados pelo sistema.

Redes Neurais Convolucionais

Rede Neural Convolucional ou, do inglês, Convolutional Neural Network(CNN). Esse tipo de rede é adequada para trabalhar com imagens, sendo capaz de atingir alto desempenho em tarefas visuais complexas. Diferente de outras redes neurais básicas, os neurônios da Rede Neural Convolucional analisam camadas anteriores em pequenos conjuntos e produzem uma saída para cada conjunto analisado. Com isso, o neurônio pode extrair mais informações dos dados ao reconhecer padrões existentes dentro dos conjuntos.
